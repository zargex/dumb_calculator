# dumb_calculator

It is an small FreePascal console calculator.

I wrote it as an example of the `getops` unit.


## How to build it

1. Make sure you have [FreePascal](https://www.freepascal.org/) installed.
2. Run `fpc dumb_calculator.pas`

## How to run it
`./dump_calculator OP arg1 arg2`

You can run `./dump_calculator -h` for help.
